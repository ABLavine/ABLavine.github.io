My Projects

ML With Python

For my final project for CS50, I made a flask website that hosted a few different neural networks and algorithms that were able to semi-accurately predict stocks, such that they turned a fair profit in the month I ran them for.

Python is my goto language if I want to get something done quickly (although that might change, as I am starting to really like Rust.) I have found that the simple syntax and its ability to do almost anything with a few imports is extremely versitle and helpful.

Automatic Work Generator

Not long ago, my history teacher mentioned the annoyingness of having to manually update every single makeup work assignment in Google Classroom. I was able to use java with the Google Classroom API in order to make a lightweight application to automate this process for her.

I would say I have a love-hate relationship with Java. While I do enjoy some of the simplicity it offers, it has many annoying quirks that stop it from being my favorite language, and so I only use it when I need to make a complileable (is that a word?) application quickly.

SQL in ML and Web

I Used SQL in the previously mentioned ML/Website project, where I needed to ensure that users had the correct monetary values, and that the relative profits had stayed the same.

I enjoy SQL, it's a very logical language once you are familiar with it, and is simple to use. Not much to say about it, to be honest. I very rarely get use of it, as the times when I need to process data in my personal life are few and far between.

JS Projects (misc)

I've used JS frequently, out of necessity. I've used it for this site, I use it to make sure the extensions I install aren't malware, I use it to make my own extensions, and I use node to run various projects like Aspine, an alternate site for students to view their grades.

However, do not mistake this frequent usage for enjoyment of the language. I don't have a love-hate relationship with JS, not an on again off again relationship, I HATE JS. It is by far my least favorite langague, and if you want me to use it, you better have a damn good reason.
